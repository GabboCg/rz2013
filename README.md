# Forecasting Stock Returns

Replication of Rapach and Zhou (2013, HEF), "Forecasting Stock Returns," *Handbook of Economic Forecasting*, Vol. 2A, Chapter 6.

## What is replicated

| Output | Script | Description |
|--------|--------|-------------|
| Table 6.1 | `table6_1.R` | R²OS and utility gain (Δ) for 14 individual predictors |
| Table 6.2 | `table6_2.R` | R²OS and Δ for combination forecasts (kitchen sink, SIC, POOL-AVG, POOL-DMSFE, diffusion index, sum-of-parts) |
| Figure 6.1 | `figure6_1.R` | Cumulative differences in squared forecast errors, individual predictors |

All results cover the 1957:01–2010:12 out-of-sample evaluation period, reported overall and separately for NBER-dated expansions and recessions.

## How to run

All scripts are R and must be run from the **repo root**.

### Step 1 — Prepare data (once)

```r
source("load.R")   # reads data-raw/handbook_data.xls → data/rz2013_data.rds
```

### Step 2 — Replicate tables and figures

```r
source("table6_1.R")    # Table 6.1: individual predictors
source("table6_2.R")    # Table 6.2: combination forecasts
source("figure6_1.R")   # Figure 6.1 → figures/figure6_1.pdf
```

## Methods

### Individual predictors (Table 6.1 / Figure 6.1)

Bivariate OLS predictive regressions of the log equity premium on each of 14 economic variables (Goyal–Welch dataset). Two forecast variants:

- **Unrestricted** — OLS forecast as-is
- **CT-restricted** (Campbell & Thompson, 2008) — set forecast to historical average when rolling slope sign disagrees with full-sample sign; floor at zero

### Combination forecasts (Table 6.2)

| Method | Description |
|--------|-------------|
| Kitchen sink | OLS on all 12 predictors simultaneously |
| SIC | BIC model selection over all 1-, 2-, 3-predictor subsets (298 models) |
| POOL-AVG | Equal-weight average of 14 individual forecasts |
| POOL-DMSFE | Discounted MSFE-weighted average (θ = 0.75) |
| Diffusion index | First PC of 14 standardised predictors |
| Sum-of-parts | Earnings growth (20-yr MA) + D/P − risk-free rate |

### Evaluation metrics

- **R²OS** (Campbell & Thompson, 2008) — percent reduction in MSFE vs. historical average benchmark; tested with Clark–West (2007) *p*-value
- **Δ** — annualised utility gain (bps) for a mean-variance investor with γ = 5, 5-year rolling variance forecast
- **CDSFE** (Figure 6.1) — cumulative difference in squared forecast errors (eq. 47); positive slope indicates outperformance of the historical average

### Key parameters

| Parameter | Value | Meaning |
|-----------|-------|---------|
| Initial window | 1926:12–1946:12 | 241 months |
| Burn-in | 1947:01–1956:12 | 120 months (combination forecasts only) |
| Evaluation | 1957:01–2010:12 | 648 months |
| γ | 5 | Relative risk aversion |
| Rolling vol window | 60 months | 5-year variance forecast |
| SOP MA window | 240 months | 20-year earnings growth MA |
| DMSFE θ | 0.75 | Discount factor |
